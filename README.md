# bvels10's OSRS Adventure Guide 💖✨

A personal, live-updating OSRS guide. Built around your real Hiscores stats — every recommendation changes as you level up.

## 🚀 How to open it

Just double-click **`start.bat`** (or open `index.html` in any browser).

That's it. No install. No setup.

## ✨ Features

- **Live stats** — pulls your levels from the OSRS Hiscores every 90 seconds
- **Next Up tab** — top 5–8 personalized recommendations based on your actual stats
- **Task list** — auto-added action items + your own. Check things off ♥
- **Tiered skill plans** — every skill shows exactly what method to use at *your* level, and when to switch
- **Boss ladder** — shows the next boss you're ready for
- **Gear progression** — the best items you can wear *right now*, by combat style
- **Quest tracker** — sorted by what's ready vs. locked. Mark them done as you go
- **RuneLite plugin list** — must-haves explained
- **Achievement Diary checklist** (Easy tier — biggest beginner unlocks)
- **Golden Rules** — common beginner mistakes to avoid
- **Keybinds cheat sheet**
- **Progress journal** — auto-logs your XP gains and level-ups
- **Personal notes pad**
- **"I'm Lost" button** — bottom-right. One click → top 3 things you should do *right now*

## 💡 Tips

- **The stats sync only updates when you log out of OSRS.** Hiscores are server-side and only reflect logged-out state.
- The username can be changed in the top-right input — useful if you want to share with a friend.
- All your task progress, journal, and notes save in your browser. They persist between sessions.
- Wiki links are sprinkled everywhere — click them when you want to go deeper.

## 🔌 Most important: install RuneLite plugins

Check the **🔌 RuneLite** tab for the full list. The single most important plugin is **Quest Helper** — it walks you through every quest in-game with arrows and a checklist. Game-changer for a new player.

## 🐛 If stats don't load

The app uses a public CORS proxy to bypass browser security. If the proxy is down:

1. Wait a minute and click the ↻ button
2. Try opening in a different browser
3. Worst case, the layout still works — you just won't see live stats
